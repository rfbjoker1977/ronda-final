# Universo Fantasy

Web estática para consultar la historia y los equipos de una liga fantasy.

## Abrir la web

Abre `index.html` con cualquier navegador moderno. No requiere instalación.

## Primer acceso de administrador

1. Pulsa **Administrador**.
2. Usa la clave inicial `cambiar-esta-clave`.
3. En el panel, pulsa **Cambiar contraseña** y define una clave personal.

Los cambios se guardan en el almacenamiento local del navegador actual. Al ser una web estática, esta protección es apropiada solo para una maqueta o uso local. Para publicar la web con edición privada real y sincronizada entre dispositivos habrá que añadir autenticación y almacenamiento en un servidor.
